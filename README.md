# DEXIAE — Lista pública de licencias revocadas

Este repositorio contiene exclusivamente el archivo `lista.json` con los hashes SHA-256 de las licencias DEXIAE que han sido revocadas.

## Propósito

La aplicación DEXIAE consulta automáticamente este archivo cada vez que se ejecuta para validar que la licencia del equipo no haya sido revocada por motivos de pago, mal uso o solicitud del cliente.

## Privacidad

Los identificadores que aparecen en la lista son **hashes SHA-256 de HWIDs**, no datos personales. No es posible identificar a una persona o equipo a partir de estos hashes.

## Uso

- **Solo lectura** para todos los clientes de DEXIAE
- **Edición** restringida al equipo de soporte de DEXIAE
